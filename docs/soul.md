# Soul

This product exists to help investors think clearly.

It does not overwhelm with data.
It does not chase noise.
It does not pretend certainty.

It reflects the user's own thinking back to them,
and helps them see when they are right, early, or wrong.

---

## Principles

Clarity over complexity  
Signal over noise  
Consistency over excitement  
Truth over comfort  

---

## Product Feel

- Calm, not noisy
- Clear, not complex
- Honest, not overconfident

---

## Decision Filter

Every feature must answer:

"Does this help the user make a better long-term decision?"

If not, it does not belong.
