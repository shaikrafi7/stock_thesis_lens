# Guardrails

## 1. No Black Box Scoring
All scores must be explainable.

---

## 2. Stable Behavior
Avoid sudden unexplained score swings.

---

## 3. Fallback Mode
System must work without:
- LLM
- External APIs

---

## 4. No Direct Financial Advice

Do NOT say:
"Sell this stock"

Say:
"Thesis weakening due to..."

---

## 5. User Control
User must always:
- See reasoning
- Adjust thesis
- Override system
